# TV Trends
###### A simple way to view the trends of TV Series

### Features
* VueJS 2 - https://vuejs.org/
* Koa 2 HTTP Container - https://github.com/zlshames/basic-koa-container
* TheMovieDB - https://themoviedb.org
* Element UI - http://element.eleme.io
* Google Charts - https://developers.google.com/chart/

### Initial Setup
```
git pull https://github.com/zlshames/tv-trends.git
cd tv-trends
yarn                                // Install dependencies
```

### Run the server
```
yarn run dev                        // Runs the development server (w/ hot reloading)
yarn run build                      // Builds front-end files for production
yarn run server                     // Start Http server container
```
